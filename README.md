# ¿Qué es una Base De Datos Relacional?
 Es una forma de organizar información en tablas (como hojas de cálculo) que se conectan entre sí para encontrar datos súper rápido y mantener el orden.

# ¿Qué es una Tabla?
 Es la estructura base donde guardas la info, organizada por filas y columnas, como una tabla de alumnos o productos.

# ¿Qué es una relación?
 Es la conexión lógica entre dos tablas, por ejemplo, conectar la tabla "Alumnos" con la tabla "Calificaciones" usando un ID compartido.

# ¿Qué representa una columna?
 Es un atributo o característica específica de los datos, como "Nombre", "Edad" o "Correo".

# ¿Qué representa un renglón?
 Es un registro único o una entrada completa, por ejemplo, todos los datos de un solo alumno.

# ¿Qué es un motor de base de datos relacional (RDBMS)?
 Es el software que te permite crear, gestionar y consultar la base de datos (ej. MySQL o SQL Server).

# Motores populares:
 MySQL, PostgreSQL, Oracle, Microsoft SQL Server, MariaDB.
# Motor ligero sin servidor:
 SQLite.

# ¿Qué significa SQL?
 Structured Query Language (Lenguaje de Consulta Estructurado). Es el lenguaje que usamos para hablarle a la base de datos y pedirle info.

# Elementos de un sistema:
 Incluye el software del motor (RDBMS), el lenguaje SQL, las tablas, los datos, y los usuarios/aplicaciones que los usan.
